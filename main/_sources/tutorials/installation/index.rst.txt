Installation
===================

pysysml is currently hosted on PyPI. It required python >= 3.8.

You can simply install pysysml from PyPI with the following command:

.. code:: shell

    pip install pysysml

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/SysMLv2-Research/pysysml@main
